arist.getPlaylinks

Returns playlinks for a list of artists

e.g. http://ws.audioscrobbler.com/2.0/?method=track.getPlaylinks&artist[]=radiohead&artist[]=Ramona+Falls&api_key=...

Params

artist[] (Required (unless mbid)] : List of artist names
mbid[] (Optional) : List of musicbrainz ids for the artists
autocorrect[0|1] (Optional) : Transform misspelled artist names into correct artist names, returning the correct version instead. The corrected artist names will be returned in the response.
api_key (Required) : A Last.fm API key.

Auth

This is a public service.
This service does not require authentication.
